# 🧘 Zen Dashboard

> **A minimalist space for deep work.**

Zen Dashboard is a distraction-free productivity tool designed to help you maintain focus and flow. It combines a customizable Pomodoro timer with a simple task list, all wrapped in a calming, atmospheric interface.

## ✨ Features

-   **Pomodoro Timer**: Customizable Focus and Break durations that fit your workflow.
-   **Task Management**: A simple, persistent checklist to track your immediate goals.
-   **Persistent Settings**: Your tasks and timer preferences (durations, last active mode) are saved automatically to LocalStorage.
-   **Zen Aesthetics**: Built with a "Vibe First" approach using dark mode, glassmorphism, and soothing ambient visuals.

## 🛠️ Tech Stack

-   **React**
-   **Tailwind CSS**
-   **Vite**
-   **Antigravity AI**

## 🚀 Setup

1.  Clone the repository.
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the development server:
    ```bash
    npm run dev
    ```
